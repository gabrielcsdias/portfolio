# Gabriel | Portfólio

Meu portfólio pessoal como desenvolvedor Full Stack.

## 🛠 Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React + React Icons
- **Fontes**: Geist + JetBrains Mono

## 🚀 Como rodar

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Rodar em produção
npm start
```

## 🌐 Deploy

Deploy automático na Vercel: **[Acessar site](https://seu-site.vercel.app)**
